## Scope of Audits

Each audit covers:
- Static analysis of Solidity source code
- Manual code review
- Formal verification (bridge contracts — Certora)
- Economic attack surface analysis
- Access control and privilege review
- Upgrade safety (UUPS proxy pattern)
- Reentrancy and overflow protection
